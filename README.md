# 全幅 / APS-C 曝光 & 景深模擬計算機

一個純前端、單檔案（HTML）即可執行的攝影計算工具。
用來快速模擬 **光圈 / 快門 / ISO** 在不同光線、不同感光元件格式下的組合，並可計算 **安全快門、等效焦距、景深與超焦距**。

開啟方式：
將 `exposure_calculator.html` 用瀏覽器打開即可。

---

## 功能摘要

### ✔ 1. EV（光線場景）預設

提供常見場景供選擇：

* EV 15：晴天直射
* EV 10：明亮室內
* EV 6：城市夜景商圈
* EV 5：街燈夜景人像
* EV 3：暗街／公園
* 也可自行輸入 EV

---

### ✔ 2. 光圈 / 快門 / ISO「自動求解」

你可以任意留空：

* 光圈 `F`
* 快門 `t`
* ISO

工具會：

* 若只留 1 個空白 → 精確解出該參數
* 若留 2 個以上 → 提供合理推薦值（避免失焦／過慢快門）

曝光公式：

```
EV100 = log2(N² / t)
ISO = 100 × 2^(EV100 − EV_scene)
```

---

### ✔ 3. 感光元件格式、等效焦距、等效景深

支援：

* Full Frame（1.0×）
* APS-C 1.5×
* APS-C 1.6×

會顯示：

* 35mm 等效焦距
* 全幅 / APS-C 等效景深（簡化版說明）

---

### ✔ 4. 手持安全快門（含防手震）

未防手震安全快門：

```
t_safe = 1 / (focal × crop factor)
```

加入防手震後：

```
t_safe_stab = t_safe × 2^(stops)
```

使用者可輸入：

* 鏡頭防手震（Lens IS）
* 機身 IBIS
* 系統會判斷你目前快門是否會手震。

---

### ✔ 5. 景深（DOF）與超焦距

輸入：

* 焦距（mm）
* 光圈 F
* 對焦距離（公尺）

即可顯示：

* 超焦距（Hyperfocal Distance）
* 近景深界限
* 遠景深界限
* 景深總長度

使用標準公式：

```
H = f² / (N × c) + f
D_near = (H × s) / (H + (s − f))
D_far  = (H × s) / (H − (s − f))
DOF = D_far − D_near
```

（s = 對焦距離、c = CoC，由全幅 / APS-C 決定）

## 技術實作

* **純前端**

  * HTML + CSS + 原生 JavaScript
  * 無框架、無依賴，適合內嵌或放在任何靜態網站
* **計算邏輯全部在前端完成**

  * 曝光計算（EV、ISO、光圈、快門）
  * 安全快門與防手震折算
  * 等效焦距
  * 景深 / 超焦距
* **UI 以卡片方式呈現結果**

  * 曝光
  * 視角 / 等效焦距
  * 安全快門 / EV 匹配
  * 景深 / 超焦距
